# Web Application Enumeration Checklist

## Application Mapping
- Identify application structure and endpoints
- Map accessible pages, parameters, and APIs
- Detect hidden or undocumented functionality

## Input and Data Handling
- Identify all user-controlled input points
- Review data validation and sanitization behavior
- Detect client-side vs server-side controls

## Authentication and Session Management
- Identify authentication mechanisms
- Review session handling and token behavior
- Detect access control weaknesses

## Common Web Vulnerabilities
- Injection points (SQL, command, template)
- Cross-site scripting vectors
- File handling and upload functionality
- Authorization and privilege boundaries

## Configuration and Exposure
- Review security headers and application configuration
- Detect exposed debug, backup, or configuration files
- Identify third-party integrations and dependencies

## Findings Documentation
- Document confirmed and potential vulnerabilities
- Assess impact and exploitation paths
- Prepare findings for reporting and remediation
